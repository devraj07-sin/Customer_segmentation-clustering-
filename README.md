📊 Customer Segmentation using K-Means Clustering

📌 Project Overview
This project focuses on segmenting customers into different groups based on their purchasing behavior using Machine Learning.
The goal is to help businesses understand their customers better and target them effectively.

🎯 Problem Statement
Businesses often struggle to identify different types of customers.
This project uses clustering techniques to group customers based on:
Annual Income
Spending Score

📂 Dataset
Dataset: Mall Customer Segmentation Data
Features used:
Annual Income (k$)
Spending Score (1-100)

🛠️ Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
Scikit-learn

⚙️ Approach
Data Loading & Exploration
Data Visualization
Feature Selection
Elbow Method to find optimal clusters
K-Means Clustering
Visualization of clusters

📈 Model Used
K-Means Clustering


📊 Results & Insights
Customers were divided into 4 clusters:
🟡 Low Spending Customers
🔵 Average Customers
🟢 High Spending Customers
🟣 Premium Customers

Key Insights:
High spending customers are ideal for marketing campaigns
Low spending customers need engagement strategies
Segmentation helps in targeted marketing
